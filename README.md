# PC Monitor

Modern lightweight desktop hardware monitor for Windows built with **WPF (.NET)** and **LibreHardwareMonitor**.

A compact floating widget that displays real-time system statistics with a clean transparent UI.

---

## ✨ Features

* 🖥 **CPU Monitoring**

  * Temperature
  * Usage %
  * Core / Thread count

* 🎮 **GPU Monitoring**

  * Temperature
  * Usage %
  * Fan speed
  * VRAM info

* 🧠 **RAM Monitoring**

  * Used memory

* 🎨 **Minimal UI**

  * Frameless transparent window
  * Always-on-top mode
  * Smooth compact design

* 📌 **System Tray Support**

  * Minimize to tray
  * Quick restore

* ⚠️ **Thermal Alerts**

  * Notifications on critical temperatures

## 🚀 Getting Started

### Requirements

* Windows 10 / 11
* .NET Desktop Runtime
* Administrator rights recommended (for full sensor access)

### Run

1. Download latest release
2. Launch `PCMonitor.exe`
3. Enjoy real-time hardware stats

---

## 📦 Build From Source

```bash
git clone https://github.com/yourusername/PCMonitor.git
cd PCMonitor
```

Open solution in **Visual Studio** and run:

```bash
Build → Release
```
